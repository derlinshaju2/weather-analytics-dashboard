# 🌡️ Weekly Temperature Analysis Dashboard
> An interactive data visualization app built with Streamlit and Matplotlib to track and compare weekly weather trends.

## ✨ Features
This application provides two primary visualization perspectives:

1.  **Maximum Temperature per Day:** A focused line chart using custom markers (`*`) and dashed lines to highlight daily peaks.
2.  **Full Temperature Comparison:** A comprehensive multi-line chart comparing **Maximum**, **Minimum**, and **Average** temperatures simultaneously to visualize the daily range.
